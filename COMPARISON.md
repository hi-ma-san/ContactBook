# ContactBook 美化效果對比

## 概述
本文檔展示使用 CSS (Qt StyleSheet) 美化前後的對比效果。

## 對比表

| 項目 | 美化前 | 美化後 |
|------|--------|--------|
| **主視窗背景** | 系統預設灰色 | 淡雅的灰藍色 (#f5f7fa) |
| **視窗標題** | "ContactBook" | "聯絡人通訊錄 Contact Book" |
| **內邊距** | 緊湊，貼邊 | 寬鬆，四周 20px |

| **表格互動** | 無特殊效果 | 選中藍色高亮，懸停淺灰背景 |
| **輸入框** | 方形，簡單邊框 | 圓角 (6px)，有內邊距，焦點時藍色邊框 |
| **標籤文字** | 系統預設 | 深色 (#24292f)，14px，中等粗細 |

| **字體** | 系統預設 | 微軟正黑體優先，更佳中文顯示 |
| **間距** | 緊湊 | 舒適 (元素間 10-15px) |
| **整體風格** | 樸素實用 | 現代專業 |

## 詳細功能對比

### 1. 視覺層次

**美化前:**
- 所有元素顏色單調
- 難以快速區分功能區域
- 視覺層次不明顯

**美化後:**
- 清晰的色彩區分
- 表格、表單、按鈕區域一目了然
- 通過顏色和間距建立視覺層次

### 2. 使用者互動回饋

**美化前:**
- 互動回饋有限
- 按鈕狀態變化不明顯
- 缺少視覺指引

**美化後:**
- 豐富的 Hover 效果
- 明確的 Focus 狀態（藍色邊框）
- 按鈕按下時顏色變深
- 表格行懸停時高亮

### 3. 功能識別

**美化前:**
- 所有按鈕外觀相同
- 需要閱讀文字才能識別功能
- 功能優先級不清晰

**美化後:**
- 綠色代表新增（積極動作）
- 橙色代表匯入（引起注意）
- 紫色代表匯出（特殊操作）
- 灰色代表結束（中性動作）
- 色彩語言幫助快速識別

### 4. 現代感

**美化前:**
- 方角設計，略顯陳舊
- 扁平但缺乏精緻度
- 傳統桌面應用外觀

**美化後:**
- 圓角設計，柔和現代
- 精心調配的配色方案
- 符合現代 UI 設計趨勢
- 類似 Web 應用的美觀度

### 5. 可讀性

**美化前:**
- 間距緊湊，略顯擁擠
- 預設字體可能不夠清晰
- 對比度一般

**美化後:**
- 充足的空白空間
- 優化的字體選擇（微軟正黑體）
- 良好的對比度
- 更大的點擊目標（按鈕 40px 高）

