# 學生聯絡簿管理系統 (ContactBook)

一個用 Qt C++ 開發的學生聯絡簿管理應用程式，提供簡單易用的介面來管理學生聯絡資訊。

## 功能特色 ✨

### 核心功能
- 📝 **新增聯絡人**：輸入學號、班級、姓名、電話等資訊
- 📥 **匯入資料**：從 CSV 或 TXT 檔案批次匯入聯絡人
- 📤 **匯出資料**：將聯絡簿資料匯出為 CSV 或 TXT 檔案
- 🗑️ **刪除資料**：刪除選中的聯絡人記錄
- ✏️ **編輯資料**：雙擊表格儲存格直接編輯
- 🔍 **排序功能**：點擊欄位標題進行排序

### 使用者體驗
- 🎨 現代化的使用者介面設計
- 💡 智慧型輸入提示與驗證
- ✅ 操作成功/失敗的即時回饋
- 🛡️ 確認對話框防止誤操作
- 🎯 自動清空輸入欄位，提升輸入效率

## 系統需求 📋

- Qt 5.x 或更高版本
- C++17 編譯器支援
- 支援平台：Windows, macOS, Linux

## 編譯方式 🔨

### 使用 qmake

```bash
qmake ContactBook.pro
make
```

### 使用 Qt Creator

1. 開啟 Qt Creator
2. 選擇 "Open Project"
3. 選擇 `ContactBook.pro` 檔案
4. 點擊建置並執行

## 使用說明 📖

### 新增聯絡人

1. 在下方表單填入資料：
   - 學號（必填）
   - 班級
   - 姓名（必填）
   - 電話
2. 點擊「➕ 新增」按鈕
3. 資料會自動加入表格中，輸入欄位會自動清空

### 匯入資料

1. 點擊「📥 匯入」按鈕
2. 選擇 CSV 或 TXT 檔案
3. 資料會自動載入到表格中

**CSV 檔案格式範例：**
```
S001,一年甲班,王小明,0912345678
S002,一年乙班,李小華,0923456789
S003,二年甲班,張小美,0934567890
```

每行包含 4 個欄位，以逗號分隔：
- 欄位 1：學號
- 欄位 2：班級
- 欄位 3：姓名
- 欄位 4：電話

### 匯出資料

1. 點擊「📤 匯出」按鈕
2. 選擇儲存位置和檔案名稱
3. 選擇檔案格式（CSV 或 TXT）
4. 資料會以 CSV 格式儲存

### 刪除資料

1. 在表格中點擊要刪除的資料列
2. 點擊「🗑️ 刪除」按鈕
3. 確認刪除操作

### 編輯資料

- 雙擊表格中的任何儲存格即可直接編輯

### 排序資料

- 點擊表格標題列（學號、班級、姓名、電話）可進行排序
