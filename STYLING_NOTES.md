# ContactBook 外觀美化說明

## 變更概述
本次更新僅針對應用程式外觀進行美化，未修改任何程式邏輯。所有功能保持不變。

## 使用的技術
- Qt Style Sheet (QSS) - 類似於 CSS 的樣式語言

## 實現的美化效果

### 1. 整體視覺風格
- **現代化設計**: 採用扁平化設計風格
- **配色方案**: 使用 GitHub 風格的藍色系為主色調
- **字體**: 優先使用微軟正黑體，提供更好的中文顯示效果

### 2. 主視窗 (QDialog)
- 背景顏色: 淺灰藍色 (#f5f7fa)
- 增加內邊距: 20px，讓內容不會貼邊
- 元素間距: 15px，提供更好的視覺呼吸空間
- 視窗標題: "聯絡人通訊錄 Contact Book"

### 3. 表格 (QTableWidget)
- 白色背景搭配圓角邊框 (8px 圓角)
- 藍色表頭 (#0969da) 搭配白色文字
- 格線顏色: 淺灰色 (#e1e4e8)
- 選中項目: 藍色高亮 (#0969da)
- Hover 效果: 滑鼠懸停時項目顯示淺灰背景
- 表頭 Hover 效果: 滑鼠懸停時顏色略深

### 4. 表單標籤 (QLabel)
- 深色文字 (#24292f)
- 字體大小: 14px
- 字重: 500 (中等粗細)
- 內邊距: 5px

### 5. 輸入框 (QLineEdit)
- 白色背景
- 圓角邊框 (6px 圓角)
- 邊框顏色: 淺灰色 (#d0d7de)
- Focus 狀態: 藍色邊框 (#0969da)
- Hover 效果: 深灰色邊框 (#8c959f)
- 內邊距: 8px 12px
- 字體大小: 14px

### 6. 按鈕 (QPushButton)
所有按鈕具有以下共同特徵:
- 圓角設計 (6px 圓角)
- 無邊框
- 內邊距: 10px 20px
- 最小寬度: 80px
- 最小高度: 40px
- 字體大小: 14px
- Hover 和 Pressed 效果

各按鈕的特色配色:
- **新增按鈕**: 綠色 (#1a7f37) - 代表新增動作
- **匯入按鈕**: 橙色 (#fb8500) - 引起注意的導入動作
- **匯出按鈕**: 紫色 (#8250df) - 特殊的導出動作
- **結束按鈕**: 灰色 (#6e7781) - 中性的退出動作

### 7. 滾動條 (QScrollBar)
- 現代化圓角滾動條設計
- 淺色背景 (#f6f8fa)
- 滾動條手柄: 灰色 (#d0d7de)
- Hover 效果: 深灰色 (#8c959f)
- 垂直滾動條寬度: 12px
- 水平滾動條高度: 12px

### 8. 間距調整
- 表單欄位水平間距: 15px
- 表單欄位垂直間距: 10px
- 按鈕之間間距: 10px

## 技術實現細節

### 樣式應用方式
在 `contactbook.cpp` 的建構函式中，使用 `this->setStyleSheet(styleSheet)` 將樣式表應用到整個對話框。這樣可以確保所有子元件都能繼承相應的樣式。

### ID 選擇器
使用 Qt 的 ID 選擇器 (如 `#insertButton`) 為不同按鈕設定不同的顏色，使其功能更加明確。

### 狀態選擇器
使用 `:hover`、`:focus`、`:pressed` 等狀態選擇器，為使用者互動提供視覺回饋。
