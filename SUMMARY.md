### 3. 技術特點

#### 實現方式
- **Qt StyleSheet (QSS)**: 類似 CSS 的樣式語言
- **Raw String Literal**: 使用 R"(...)" 語法便於編寫多行樣式
- **集中式管理**: 所有樣式在一處定義
- **繼承式應用**: 在父元件應用，子元件自動繼承

#### 選擇器使用
- **類型選擇器**: `QDialog`, `QTableWidget`, `QLineEdit`
- **ID 選擇器**: `#insertButton`, `#exportButton`
- **狀態選擇器**: `:hover`, `:focus`, `:pressed`
- **子元素選擇器**: `QTableWidget::item`, `QHeaderView::section`

#### 配色系統
- **主色**: GitHub 藍 (#0969da) - 專業、可信
- **成功**: 綠色 (#1a7f37) - 積極、新增
- **警告**: 橙色 (#fb8500) - 引起注意
- **資訊**: 紫色 (#8250df) - 特殊、導出
- **中性**: 灰色 (#6e7781) - 退出、取消
- **背景**: 淺灰藍 (#f5f7fa) - 柔和、舒適
