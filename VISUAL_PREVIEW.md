# ContactBook 視覺效果預覽

## 整體外觀描述

### 應用程式視窗
- **視窗標題**: "聯絡人通訊錄 Contact Book"
- **視窗大小**: 800x600 像素
- **背景顏色**: 淺灰藍色 (#f5f7fa)，提供柔和的視覺體驗
- **內邊距**: 四周各 20px，內容不會緊貼邊緣

## 各部分詳細說明

### 1. 表格區域 (頂部)
```
┌────────────────────────────────────────────────────────────────┐
│ ╔════════════════════════════════════════════════════════════╗ │
│ ║  學號    │    班級    │    姓名    │    電話           ║ │ <- 藍色表頭 (#0969da)
│ ╠════════════════════════════════════════════════════════════╣ │
│ ║                                                            ║ │
│ ║                  (表格內容區域)                             ║ │ <- 白色背景
│ ║                                                            ║ │
│ ║                                                            ║ │
│ ╚════════════════════════════════════════════════════════════╝ │
└────────────────────────────────────────────────────────────────┘
```

特點:
- 白色背景，8px 圓角
- 藍色表頭文字為白色，字體加粗
- 滑鼠懸停在表頭上時顏色變深
- 選中的行顯示藍色背景
- 滑鼠懸停在行上顯示淺灰色背景
- 細緻的格線分隔

### 2. 表單輸入區域 (中間)
```
學號：  [                    ]  <- 輸入框，白色背景，圓角邊框
班級：  [                    ]
姓名：  [                    ]
電話：  [                    ]
```

特點:
- 標籤文字: 深色 (#24292f)，14px，中等粗細
- 輸入框: 白色背景，灰色圓角邊框
- 獲得焦點時: 藍色邊框
- 滑鼠懸停時: 深灰色邊框
- 內邊距充足，容易輸入

### 3. 按鈕區域 (底部)
```
     [  新增  ]      [  匯入  ]      [  匯出  ]      [  結束  ]
      綠色           橙色           紫色           灰色
```

按鈕顏色說明:
- **新增** (insertButton): 綠色 (#1a7f37)
  - 代表創建新項目的積極動作
  - Hover: 稍微變亮
  - Pressed: 深綠色 (#116329)

- **匯入** (pushButton): 橙色 (#fb8500)
  - 引起注意的導入動作
  - Hover: 稍暗的橙色 (#e07700)
  - Pressed: 深橙色 (#c66900)

- **匯出** (exportButton): 紫色 (#8250df)
  - 特殊的導出動作
  - Hover: 稍暗的紫色 (#7745cc)
  - Pressed: 深紫色 (#6639ba)

- **結束** (exitButton): 灰色 (#6e7781)
  - 中性的退出動作
  - Hover: 稍深的灰色 (#57606a)
  - Pressed: 深灰色 (#424a53)

所有按鈕特點:
- 6px 圓角，現代化外觀
- 無邊框，扁平化設計
- 白色文字
- 最小高度 40px，易於點擊
- 最小寬度 80px
- 按鈕之間有 10px 間距
- 滑鼠互動時顏色變化流暢

### 4. 滾動條 (當內容過多時)
```
║ ║  <- 圓角滾動條
║█║
║ ║
```

特點:
- 12px 寬度 (垂直) 或高度 (水平)
- 淺色背景 (#f6f8fa)
- 深色滾動把手 (#d0d7de)
- 滑鼠懸停時滾動把手變深 (#8c959f)
- 圓角設計，與整體風格一致

## 配色方案總結

### 主要顏色
- **主色調**: 藍色 (#0969da) - 表頭、選中狀態、焦點邊框
- **背景色**: 淺灰藍 (#f5f7fa) - 主視窗背景
- **內容背景**: 白色 (#ffffff) - 表格、輸入框

### 功能顏色
- **成功/新增**: 綠色 (#1a7f37)
- **警告/匯入**: 橙色 (#fb8500)
- **資訊/匯出**: 紫色 (#8250df)
- **中性/結束**: 灰色 (#6e7781)

### 邊框和分隔
- **標準邊框**: 淺灰色 (#d0d7de)
- **格線**: 更淺的灰色 (#e1e4e8)
- **懸停邊框**: 深灰色 (#8c959f)

### 文字顏色
- **主要文字**: 深色 (#24292f)
- **高亮文字**: 白色 (在深色背景上)

## 使用者體驗改進

1. **視覺層次清晰**: 通過顏色和間距區分不同功能區域
2. **互動回饋明確**: Hover 和 Pressed 狀態提供即時視覺反饋
3. **功能識別容易**: 不同顏色的按鈕幫助使用者快速識別功能
4. **現代化設計**: 圓角、扁平化風格符合現代審美
5. **易讀性高**: 充足的間距和適當的字體大小
6. **無障礙友好**: 足夠的對比度和大小的點擊目標

## 對比: 美化前 vs 美化後

### 美化前
- 灰色單調的預設外觀
- 方形邊角，略顯生硬
- 按鈕無色彩區分
- 間距緊湊
- 預設系統字體

### 美化後
- 現代化的藍色系配色
- 圓角設計，柔和親和
- 彩色按鈕系統，功能明確
- 舒適的間距布局
- 優質的中文字體
- 豐富的互動效果

## 技術特點

- **純 CSS (QSS) 實現**: 無需修改程式邏輯
- **可維護性高**: 樣式集中管理
- **跨平台一致性**: Qt 保證不同平台外觀統一
- **性能優化**: 樣式在初始化時一次性應用

## 總結

這次美化完全遵循了「不要修改程式邏輯」的要求，僅通過 Qt StyleSheet 為應用程式帶來了現代化、專業的視覺效果。所有功能保持不變，使用者可以獲得更好的視覺體驗和操作體驗。
